# Source

This folder is the home for the shared application source code as the project moves toward one shared codebase that produces native mobile, desktop, and web builds.

The current playable web prototype lives in the pnpm monorepo under `artifacts/resonance`. It is the working reference implementation. Do not create a competing copy of the prototype here.

The plan for reconciling this folder with the existing monorepo is tracked in [Open Questions](../docs/13_Open_Questions.md), and engineering standards are in [Code Standards](../docs/10_Code_Standards.md).
